## ğŸ§  Theory Overview

The Mohrâ€“Coulomb criterion defines failure as occurring when the Mohr circle touches or exceeds a straight envelope defined by:

\[
\tau = c + \sigma \cdot \tan(\phi)
\]

Where:
- \( \tau \): shear stress  
- \( \sigma \): normal stress  
- \( c \): cohesion  
- \( \phi \): internal friction angle
